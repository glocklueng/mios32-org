Required hardware:
   o MBHP_CORE_LPC17
   o up to 4 MBHP_DINX4 modules
   o up to 4 MBHP_OUTX4 modules
   o a SD Card to store the customized setup (and to load MIDI files)

Optional hardware:
   o a 2x20 LCD for configuration + 6 buttons + 1 rotary encoder
